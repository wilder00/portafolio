Las 2 fases para instalar wordpress en entorno local

Wordpress es un Sistema de gestion de contenido (CMS) que te permite administrar sitios web en una con una vista o interface hecha para el usuario y le sea fácil de manejar. El objetivo de los CMS es que se le facilite al usuario la administración de su sitio web ya que hay usuarios que tiene como objetivo y prioridad la creación de contenido web de forma sencilla y por ello rápida sin lidiar con la parte de la programación o los códigos. Existe servicios en internet que te ofrecen directamente acceso a una cuenta wordpress para que directamente solo te pongas a crear contenido es el caso de WordPress.com que te permite crearte una cuenta gratuita con algunas limitaciones.

Para personas nuevas como yo, tal vez sea necesario interactuar con su interface para probarlo y estudiarlo de ser necesario. Esto es posible gracias a que su código está a disposición de cualquiera que desee estudiarlo o instalarlo en su servidor web. Por ello dejo esta guía para el que desee instalarse wordpress funcionando en su computadora y para mis futuras consultas.

FASE 1: Instalar XAMPP

Como primer paso necesitamos de un entorno que soporte (es decir que ayude a que wordpress funciones), para ser exactos, se necesita de un software que sirva como servidor web y otro que sea de base de datos. Sin embargo, la instalación de esos servicios se nos facilita gracias a XAMPP, un software que viene con el entorno necesario para usar wordpress. Sus iniciales nos indican los que se va a instalar. La "X" indica el sistema operativo en la que está disponible y quiere decir que funciona en cualquiera de la que conocemos. La "A" indica Apache que es un software que funcionará como servidor web. La "M" viene de MariaDB que es similar a MySql y es un gestor de base de datos. Finalmente las dos "P", el primero indica PHP que es un lenguaje de programación con la que fué hecha Wordpress; la última "P" es de Perl, otro lenguaje de programación que no es indispensable para la instalación de wordpress.

Lo primero que debemos hacer es descargar el instalador de XAMPP. En el sitio web de XAMPP Puedes descargarlo para Windows, Linux y Mac.

Una vez descargado:

  1. Ejecutamos el archivo descargado como administrador o simplemente con doble click que nos pedirá permisos para ejecutarlo.
  2. El instalador puede darnos alguna advertencia, le damos click en OK.
  3. El instalador les dará la bienvenida, le dan click en siguiente (next).
  4. Puedes dejar seleccionado todo y dar en next, en mi caso solo mantendré seleccionado los siguientes:
  5. El instalador nos da la opción de cambiar la ruta de instalación. Dejaré la ruta de instalación por defecto C:\xampp y next.
  6. Dejaré el lenguaje por defecto y luego next.
  7. Deseleccionare Learn more about Bitnami for XAMPP y luego next.
  8. El instalador nos muestra una mensaje indicando que la instalación va a comenzar después de dar next
  9. Tomará una poco de tiempo instalarse, pero durante la instalación saldrá una alerta de seguridad y le damos en Permitir acceso.
  10. Una vez terminado le das en finalizar (finish) ¡Y ya tenemos XAMPP!

Para probar que todo esté correcto, vamos a correr los dos servicios escenciales, el servidor web y el gestor de base de datos. Daremos click al botón start de Apache y MySql. Este debería hacer que en el panel de control de XAMPP se resalten de verde las palabras Apache y MySql. Después, daremos click en el boton Admin de MySql. El panel de control abrirá el navegador con el enlace: http://localhost/phpmyadmin/ El enlace apunta al servidor en tu computadora, por lo que si puedes ver el panel de PHPMyAdmin, entonces todo está correcto.

FASE 2: Instalar Wordpress

Ahora comenzamos con el objetivo principal de este post. Así que lo primero es descargar wordpress de su sitio web wordpress.org. Como podemos observar, existe wordpress con .org y .com, por ello debemos fijarnos que estemos en el .org porque el otro es wordpress alojado para crearse cuenta y no de la descarga.

Creando una nueva base de datos

En lo que se descarga vamos a configurar algunos elementos necesarios para que wordpress pueda ejecutarse. Nos tenemos que dirigir al panel de administración de PHPMyAdmin

  1. En el panel de PHPMyAdmin damos click a Nueva.
  2. Luego ponemos un nombre a la base de datos (el que desees), en mi caso le puse wilder_wordpress.
  3. Finalmente dar click en crear.

Configurando el Wordpress

Después que haya terminado la descarga del archivo de wordpress nos quedará un archivo comprimido en formato .zip. Con ese detalle claro, seguirá lo siguiente:

  1. El comprimido lo tienes que llevar a la ruta donde instalaste xampp, en mi caso fue C:\xampp por ello, estoy copiando el archivo comprimido en la carpeta de dirección C:\xampp\htdocs.
  2. Ahora, pasamos a descomprimirlo. Ello nos dará como resultado una carpeta llamada "wordpress", puedes dejarlo como está o cambiarle de nombre. En mi caso, le cambiaré a wilder.
  3. Después, necesitaremos entrar a la carpeta de wordpress que descomprimimos y encontrar el archivo llamado wp-config-sample.php
  4. Tenemos que cambiarle el nombre del archivo de wp-config-sample.php a wp-config.php, es decir quitarle el -sample.
  5. Ese mismo archivo lo tenemos que abrir con algún editor de texto y porner la información de la base de datos que creamos en PHPMyAdmin.
    • En la linea 23, ponemos en el segundo parámetro el nombre que le dimos a la base de datos que creamos, en mi caso fue wilder_wordpress
    • En la linea 26, ponemos en el usuario de la base de datos, en nuestro caso no lo creamos, pero por defecto xammp crea root
    • En la linea 29, ponemos la constraseña del usuario, el usuario que se crea con XAMPP no tiene constraseña así que se deja vacío con las comillas simple''
    • En la linea 32, 35 y 38, lo dejamos como está
  6. Ahora nos toca poner los SALT Keys que genera automáticamente Worpress con su API en http://api.wordpress.org/secret-key/1.1/salt
    • De la linea 49 al 56, se completa con el API generador de salt de wordpress.org. Cada Salt que se genera con su api es distinta así que se sustituye toda esas lineas como se muestra en la imagen siguiente.
  7. Finalmente, guardas las configuraciones y te vas al navegador y pones la url http://localhost/wilder en mi caso es wilder porque le cambié el nombre de la carpeta de wordpress al descomprimirla. Al darle enter, debería aparecer el formulario para registrarte como usuario y administrador del sitio web.

Registrandose en wordpress

Solo nos queda registrarnos en wordpress para empezar a crear y editar contenido en nuestro entorno local. al ir a la url mencionada anteriormente, que en mi caso es http://localhost/wilder saldrá formulario para que te registres fácil de seguir.

Rellenas cada campo con lo que creas conveniente y después de darle al botón install wordpress te pedirá que inicies sesión con los credenciales del registro y con eso ya está instalado wordpress.
