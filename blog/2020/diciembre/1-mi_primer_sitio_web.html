Mi primer sitio web

Hoy en día muchas de las personas que crecen profesionalmente por lo menos han tenído algun lugar en internet en la que expresan sus ideas y conocimientos. Existen herramientas que facilitan ese proceso comenzando por las redes sociales, sin embargo, algo que sirve bien para centrar todo esos medios por las que te expresas es un sitio web. Por ello, ante la necesidad de dejar una huella en internet sobre mí, nació la idea de crear mi sitio web, pero ¿qué tan dificil fue? Este es una pregunta que me es dificil responder, ello se debe a que es el primer sitio web que hago y termina publicado, además sigo en el proceso de experimentar lo que que conlleva administrar uno como principiante. En este post explicaré de manera breve parte de mi experiencia al relizar este website.

¿Cómo empecé?

No es algo complicado hacerlo si pasas por algunos cursos de html, css y javascript, lo complicado está en empezar. Te puedes ver varios cursos relacionados de creacion de paginas web, pero si no tienes una motivación se complica. En mi caso, cuando pensé en crear este sitio web lo estuve aplazando cada día buscando algún motivo. Alguno de ellos fueron como: "cuando acabe mis clases de este ciclo", "ya en las vacaciones", "por hoy voy a descansar" y demás excusas que para el momento servian para engañar a mi mente. y funcionaban hasta que un día me puse a reflexionar sobre esa repetición.

Después de pensar en lo que sucedería acabando el semestre, supe que no lo iba a hacer si no lo empezaba ¡ya! asi que me puse a escribir alguno códigos de html, los básicos, como las etiquetas básicas del <head></head> y del <body></body> imaginandome algunos componentes de una página básica como la barra de navegación con el <nav></nav> y el <footer></footer> muy básico que hasta ahora sigue así. La idea era empezar con algo aunque estuviera mal que después se pueden corregir los errores. Además, el empezar a escribir el código te impulsa a investigar y darte cuenta de detalles que creias que podías manejar cuando llegue el momento de empezar, pero, como en mi caso, terminas buscando en google.

Busqueda de referencias

No soy experto en diseño, de hecho soy principiante en todo. Asi que, una siguiente fase fue buscar referencias en distintos sitios web, pero no lo hice de manera forzada. Al contrario, en lo que leía los posts de recomendaciones de google (porque google nos conoce), iba viendo cómo estaban distribuidos algunos de los elementos de su website y también su estilo o diseño, algunos comportamientos que tenían. De esa manera iba imaginando cómo podría quedar mi pequeño website Frankenstein (¿entienden? ¿Frankenstein, partes? yo sé que sí). Tomé varias capturas de pantalla que las guardo como inspiración para futuras implementaciones o modificaciones.

De por sí, la misma recomendación tiene partes que te pueden servir de ideas para implementarlas en tu sitio web, los cards. Estos tienen mucha utilidad ya que los cards que realices los pudes reutilizar. Si solo quieres practicar lo que vayas aprendiendo cuando realices cursos de html y css, la mejor forma es construyendo diferentes cards. Mientras vayas intentando hacer algo parecido a tus referencias te irás preguntando ¿cómo puedo hacer algo parecido a eso? y ello te llevará a investigar y aplicarlo en tu proyecto (eso fue lo que me pasó).

La publicación

Para publicarlo en internet, no fue más que utilizar las herramientas de github. Por ello, te recomiendo saber git y github, conocimientos básicos. Este sitio web se encuentra alojado en github gracias a github pages. Este funciona como un servidor estático para que puedas mostrar a todos, tus archivos html, css y javascript para que sean renderizados en los navegadores, realmente es muy útil. Probablemente haya otras herramientas más, pero esta es la única que conozco más fácil de acceder.

A modo de conclución, si estás tomando cursos de las tecnologías básicas de la web como html y css ,será una buena base para que puedas crear tu sitio web. Te sugiero practicar creando cards y, si tienes la oportunidad, revisa los sitios web que utilizarás como referencia. El inspector de elementos que trae el navegador ayuda mucho para ver cómo están construido. En mi caso, como dije, soy aún un principiante y mi nivel llega a html, css y javascript básico. Así que, si estás igual que yo y aún no hiciste un proyecto un poco más grande como un sitio web, te animo a que lo intentes, verás que en el proceso surgirán preguntas que no tomaste en cuenta y si te pones a investigar para resolver esas dudas aprenderás más.
