Primera entrada blog

Un par de meses atrás decidí crear un sitio web desde cero con la información que podía encontrar por internet. Estuve creandolo de a poco hasta que llegó la parte de hacer el área de las entradas para el blog. por ello, necesitaba de uno para hecharle los estilos y ver cómo iba quedando. Como base este es el indicado.

Lo que terminaré de hacer es un diseño básico que me permita iniciar mis publicaciones de esta manera no se dilata el tiempo de este proyecto personal y con ello reduciría la propabilidad de abandolarlo. Lo más probable es que termine con varios errores ya que este es el primer proyecto hecho de cero que quiero publicar si o si.

De momento espero que con lo que vaya escribiendo sea una buena referencia para otros post. Utilizaré este primer post como una plantilla para las demás. De esta manera tendré algo de libertad para insertar y modificar lo que desee y para lo que quiera experimentar. La única difucultad es que solo será por la parte front end sin algún tipo de CMS. Por ello, probaré algunas opciones como plugins para sitios web como las de comentarios de facebook y los botones relacionados a los likes, este sería la unica forma de obtener feedback directo de los lectores.
